# Kahoot Bot

https://github.com/Jones8683/Kahoot-Bot

## Setup

&nbsp;&nbsp;&nbsp; **1. Clone repository**

```bash
git clone https://github.com/Jones8683/Kahoot-Bot
```

&nbsp;&nbsp;&nbsp; **2. Install dependencies**

```bash
npm install
```

## Run (Numbered Names)

```powershell
node index.js GAME_PIN 3 name
```

Creates:

- name 1
- name 2
- name 3

## Run (Custom Names)

```powershell
node index.js GAME_PIN --names "lucas,mia,noah"
```

## Easy Run Mode

Double-click `run.bat`
